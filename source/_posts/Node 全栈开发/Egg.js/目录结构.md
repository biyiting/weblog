## 环境准备
1. 操作系统：支持 macOS，Linux，Windows；
2. 运行环境：建议选择 LTS 版本，最低要求 8.x；

## 快速初始化
1. 推荐直接使用脚手架，只需几条简单指令，即可快速生成项目（npm >=6.1.0）:
    ```shell
    $ mkdir egg-example && cd egg-example
    $ npm init egg --type=simple
    $ npm i
    ```
2. 启动项目:
    ```shell
    $ npm run dev
    $ open http://localhost:7001
    ```
