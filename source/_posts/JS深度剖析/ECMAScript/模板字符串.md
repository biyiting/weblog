## 模板字符串特点：
1. 反引号包裹；
2. 允许换行；
3. 可使用插值表达式；

## 字符串扩展方法
```JS
const message = 'Error: foo is not defined.'

// 判断是不是 Error 开头
console.log(message.startsWith('Error')) //true

// 判断是不是 . 结尾
console.log(message.endsWith('.')) //true

// 判断是不是包含 foo
console.log(message.includes('foo')) //true
```
